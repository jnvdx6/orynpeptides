# ORYN BRANDBOOK
## Precision Peptide Science

---

## 1. BRAND CORE

**Name:** ORYN
**Pronunciation:** /OR-in/
**Tagline:** Precision Peptide Science
**Category:** European Biotech Laboratory

### Brand Promise
ORYN delivers research-grade peptide solutions with pharmaceutical-level manufacturing standards, engineered for molecular precision.

### Brand Personality
- Archetype: Scientist + Engineer
- Tone: Authoritative, precise, clean, trustworthy
- Style: Minimal futurism, laboratory premium

### ORYN Must Feel Like:
- A serious European biotech laboratory
- Advanced pharmaceutical research
- Premium hardware precision (Apple aesthetic)
- Clean, clinical, trustworthy

### ORYN Must NEVER Feel Like:
- Fitness/supplement brand
- Biohacking/wellness lifestyle
- Discount or bulk supplier
- Informal or unscientific

- **10%** ORYN Orange (#FF6A1A) - accents, CTAs, highlights

> The orange is the protagonist but used sparingly. It must feel like a scientific accent, not a brand color flood.

## 3. TYPOGRAPHY

### Primary: Space Grotesk
- Geometric, scientific, modern
- Used for: headings, navigation, UI elements, brand text
- Weights: 300 (Light), 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)

### Secondary: IBM Plex Sans
- Industrial, clean, readable
- Used for: body text, descriptions, long-form content
- Weights: 300, 400, 500, 600, 700

### Monospace: IBM Plex Mono
- Technical, laboratory feel
- Used for: specs, data, labels, batch numbers, codes
- Weights: 400, 500

### Type Scale
- Hero: 72px / Bold / Space Grotesk / Tracking: -0.02em
- H1: 60px / Bold / Space Grotesk / Tracking: -0.01em
- H2: 40px / Bold / Space Grotesk
- H3: 24px / Bold / Space Grotesk
- Body Large: 20px / Regular / IBM Plex Sans / Leading: 1.6
- Body: 16px / Regular / IBM Plex Sans / Leading: 1.6
- Small: 14px / Regular / IBM Plex Sans
- Caption: 12px / Regular / IBM Plex Mono / Tracking: 0.05em
- Label: 10px / Bold / IBM Plex Mono / Tracking: 0.2em / Uppercase

## 4. LOGO

### Wordmark: ORYN
- Font: Space Grotesk Bold
- Letter-spacing: 0.2em (expanded)
- The O incorporates a molecular ring with orange accent dots
- Reads as: scientific, precise, modern

### Icon Mark
- Square with rounded corners containing a molecular ring circle
- Two orange (#FF6A1A) accent nodes at 12 and 6 o'clock positions
- Functions as: favicon, app icon, vial label, packaging mark

### Logo Variations
1. **Full Logo**: Icon + ORYN wordmark + "PEPTIDE LABORATORY" sub-text
2. **Compact**: Icon + ORYN wordmark
3. **Icon Only**: Square molecular ring mark
4. **Wordmark Only**: ORYN text with orange accent dot

### Clear Space
- Minimum clear space around logo = height of the "O" character
- Logo must never be placed on busy backgrounds without sufficient contrast

### Color Variations
- Dark on light: #121212 on #FAFAFA
- Light on dark: #FAFAFA on #121212
- Orange accent dots always remain #FF6A1A

## 5. VISUAL ELEMENTS

### Molecular Grid
- Subtle background pattern of 60x60px grid lines
- Light: rgba(217, 217, 217, 0.3) on white
- Dark: rgba(255, 255, 255, 0.03) on black
- Creates laboratory/scientific atmosphere

### Accent Line
- 40px x 2px horizontal orange line
- Used as section dividers and card accents
- Can be vertical (2px x 40px) for sidebar elements

### Status Dots
- Small colored dots (2-3px) used as decorative elements
- Orange (#FF6A1A): active, highlight, accent
- Grey: separator between specs

### Scientific Imagery Style
- Molecular structures and bonds as decorative overlays
- Abstract peptide chains
- HPLC chromatography patterns
- DNA double helix references
- Always subtle, never overpowering

### Photography Style
- Premium product photography (Apple aesthetic)
- White or light grey surfaces with soft shadows
- Black matte product finishes
- Orange accent lighting
- Shallow depth of field
- Clean, clinical environments
- ISO cleanroom and laboratory settings

## 8. DIGITAL DESIGN

### Website
- Ultra-minimal layout
- 60% white space
- Max content width: 1280px (max-w-7xl)
- Grid: 12-column responsive
- Border radius: 12px (rounded-xl) for cards, 16px (rounded-2xl) for sections
- Shadows: subtle, warm-toned

### Buttons
- Primary: Black bg, white text, rounded-xl, hover -> orange
- Secondary: White bg, border, hover -> orange border
- CTA: Orange bg, white text, shadow

### Cards
- White bg, 1px grey border, rounded-2xl
- Hover: translateY(-8px), larger shadow
- Orange accent line on hover

### Animations
- Entrance: fadeIn 0.8s, slideUp 0.8s
- Hover transitions: 0.3s-0.4s cubic-bezier(0.16, 1, 0.3, 1)
- Molecular ring rotation: 20s linear infinite
- Pulse effects: 4s for status dots
